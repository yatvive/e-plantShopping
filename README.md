# e-plantShopping
# coding-project-template
Dans ce projet final, nous créer une application de panier d'achat pour une boutique en ligne de plantes qui propose une variété de plantes d'intérieur.

Les fonctionnalités du panier d'achat Paradise Nursery incluront :

Une page d'accueil avec un bouton liant à la page de liste de produits
Une barre de navigation avec des liens vers les pages d'accueil, de liste de produits et de panier
Une carte pour chaque plante qui présente les différentes plantes avec leurs images, nom, description, coût et un bouton Ajouter au panier.
Un minimum de deux sections décrivant les plantes dans cette section. Par exemple, "Plantes aromatiques" et "Plantes médicinales".
Une page de panier qui affiche les produits dans le panier.
Le panier doit avoir une carte pour chaque type de plante dans le panier. Chaque carte doit avoir la miniature, le coût unitaire, le coût total de toutes les plantes de ce type et des boutons pour augmenter et diminuer la quantité ainsi qu'un bouton Supprimer.
Un bouton Continuer vos achats et Passer à la caisse.
